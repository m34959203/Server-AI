# Задачи для AI-Server

## Приоритет 1 — Критические (исправить сейчас)

### 1. Исправить голосовой ввод (STT)
**Что:** Whisper не принимает аудио от Open WebUI (HTTP 400)
**Команда для Claude Code на сервере:**
```
проверь логи whisper и voice-assistant, найди причину ошибки Transcription failed и исправь
```

### 2. Настроить HTTPS (SSL)
**Что:** Браузер блокирует микрофон без HTTPS
**Команда:**
```
настрой Tailscale HTTPS или Caddy reverse proxy для сервисов на портах 3000, 3002, 8188
```

## Приоритет 2 — Интеграции

### 3. Интегрировать TTS (Piper)
**Что:** Подключить синтез речи на русском к Open WebUI
**Команда:**
```
настрой Piper TTS в Open WebUI, проверь что русский голос irina работает
```

### 4. Настроить RAG (загрузка документов)
**Что:** Загрузить документы университета в Open WebUI для поиска по ним
**Команда:**
```
настрой RAG в Open WebUI с nomic-embed-text, загрузи тестовый документ
```

### 5. Настроить Perplexica (AI-поиск)
**Что:** Развернуть поисковый движок с AI
**Команда:**
```
добавь Perplexica в docker-compose.yml и настрой подключение к Ollama
```
