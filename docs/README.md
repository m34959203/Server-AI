# Документация AI-Server

## Структура папок

| Папка | Содержимое |
|-------|------------|
| `instructions/` | Пошаговые инструкции по настройке сервисов |
| `guides/` | Руководства для пользователей и администраторов |
| `configs/` | Резервные копии конфигураций и примеры |
| `logs/` | Журналы работ и изменений |
